<h2><a href="https://www.geeksforgeeks.org/problems/inorder-successor-in-bst/1">Inorder Successor in BST</a></h2><h3>Difficulty Level : Difficulty: Easy</h3><hr><div class="problems_problem_content__Xm_eO" style="--text-color: var(--problem-text-color);"><p><span style="font-size: 14pt;">Given a BST, and a reference to a Node <strong>k</strong> in the BST. Find the Inorder Successor of the given node in the BST. If there is no successor, return -1.&nbsp;</span></p>
<p><span style="font-size: 14pt;"><strong>Examples :</strong></span></p>
<pre><span style="font-size: 14pt;"><strong>Input:</strong> root = [2, 1, 3], k = 2
<strong>Output: </strong>3 
<strong>Explanation:</strong> Inorder traversal : 1 2 3 Hence, inorder successor of 2 is 3.<br><img src="https://media.geeksforgeeks.org/img-practice/prod/addEditProblem/923613/Web/Other/blobid5_1777396060.jpg" height="100"><br></span></pre>
<pre><span style="font-size: 14pt;"><strong>Input:</strong> root = [20, 8, 22, 4, 12, N, N, N, N, 10, 14], k = 8 &nbsp; &nbsp; 
<strong>Output: </strong>10<strong>
Explanation: </strong>Inorder traversal: 4 8 10 12 14 20 22. Hence, successor of 8 is 10.<br><img src="https://media.geeksforgeeks.org/img-practice/prod/addEditProblem/923613/Web/Other/blobid3_1777396006.jpg" width="165" height="199"></span></pre>
